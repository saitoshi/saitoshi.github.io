When I was making my plans for this semester, I had my doubts. I was warned about a semester with both ICS 311: Algorithms and ICS 314: Software Engineering. My peers warned me that it is going to be a dangerous semester. Now I understand why so many of my peers warned me about. ### Initial thoughts on JavaScript ###

After a week with tweaking with JavaScript, the initial thought I had was that this is not like any other language I learned before. The first two programming languages I learned was C and Java. These two languages while having their differences, they had more things in common compared to JavaScript. 

To begin with, the variable declarations are different. For example, C and Java has declaration for digits such as integers, float, or doubles. JavaScript, on the other hand the variable declaration are not always necessary. However, the for loop structure and if - else statements are not different from the other languages, but since it is a foreign language, there is a brain barrier that makes learning difficult. Furthermore, there were many online lessons that made the task of learning JavaScript a tedious task.
